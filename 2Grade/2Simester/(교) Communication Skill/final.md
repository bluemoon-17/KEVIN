## 📝 10주차: 관계의 문을 여는 소통의 기술 - 갈등 상황 대처 상세 요약 (시험 대비)

### 1. 갈등 상황의 인식과 대처

| 구분 | 주요 개념 | 상세 설명 및 특징 |
| :--- | :--- | :--- |
| **갈등의 정의** | **정의** | 서로 다른 목표, 관심사, 의견, 가치관 등이 충돌하여 발생하는 대립 상태. <br> 특히, 인간관계에서 갈등은 **나의 욕구가 좌절되었을 때** 감정적 반응으로 나타나며, 습관적이고 공격적인 표현으로 상황을 악화시킬 수 있습니다. |
| **갈등 상황의 반응** | **습관적/자동적 반응** | 갈등 상황에서 나오는 말들은 대부분 **습관적**이고 **자동적**인 경우가 많으며, 상대방을 질책하거나 공격하는 방식으로 표현되어 관계를 악화시킵니다. (예: "너 때문에 이런 일이 벌어진 거야!") |
| **갈등의 긍정적 측면** | **숨겨진 메시지** | 갈등 상황에서 나오는 날카로운 말(예: "너랑은 절교야!") 속에는 "나 좀 알아줘. 내 마음이 어떤지…"와 같은 **욕구 좌절의 메시지**가 숨어 있습니다. <br> 갈등은 **친밀감**을 높일 수 있는 **성장의 기회**가 되기도 합니다. |
| **갈등 대처 훈련** | **필요성** | 갈등 상황에서는 감정을 분리하고 **존중의 대화**를 통해 기분대로 반응하지 않도록 훈련해야 합니다. |

### 2. 소통의 기술: 나 전달법 (I-message)

| 구분 | 주요 개념 | 상세 설명 및 구성 요소 |
| :--- | :--- | :--- |
| **나 전달법 (I-message)** | **정의** | 상대방을 비난하거나 평가하지 않고, 상대방의 행동이 **나에게 미친 영향과 나의 감정**을 주어 '나'를 사용하여 전달하는 화법입니다. |
| **너 전달법 (You-message)** | **정의** | 주어 '너(You)'를 사용하여 상대방의 **잘못을 비난**하거나 **평가**하는 화법입니다. (예: "너는 왜 스마트폰만 보고 있어?") <br> **결과**: 상대방에게 분노, 저항감, 반발심을 유발하고 방어적으로 만들며, 문제 해결을 방해합니다. |
| **나 전달법의 3단계 구성 요소** | **① 상대방의 행동에 대한 비난 없는 서술** | 내가 관찰한 상대방의 행동을 **사실 그대로 중립적**으로 이야기합니다. (예: "네가 스마트폰을 계속 보고 있으니까...") |
| | **② 상대방의 행동이 나에게 미친 감정** | 그 행동 때문에 내가 **느낀 감정**을 표현합니다. (예: "...약간 초조하고 답답해...") |
| | **③ 상대방이 해주기를 바라는 사항** | 내가 원하는 **구체적인 행동**을 부탁합니다. (예: "...스마트폰을 잠시 두고 같이 발표 준비에 집중해 주면 정말 고마울 것 같아.") |
| **나 전달법의 효과** | **장점** | 상대방에게 비난이 아닌 **정보**를 전달하여, 상대가 **방어하지 않고** 나의 이야기를 경청하게 만듭니다. <br> 상대방에게 **행동 변화의 책임**을 부여하며, 문제를 **협력적**으로 해결할 수 있는 기반을 마련합니다. |


## 📝 11주차: 비폭력 대화(NVC) I 상세 요약 (시험 대비)

### 1. 삶을 소외시키는 대화 (폭력적 대화 방식)

| 구분 | 주요 개념 | 상세 설명 및 특징 |
| :--- | :--- | :--- |
| **정의** | **삶을 소외시키는 대화** | 사람들을 서로 **폭력적으로 말하고 행동하게** 하는 대화 방법으로, 관계를 악화시키는 핵심 요소입니다. |
| **① 도덕주의적 판단** | **특징** | 내가 가진 선입견, 편견, 가치관을 기준으로 상대를 판단하고 비난하는 대화입니다. (예: "너는 너무 이기적이야", "그것은 잘못된 생각이야"). |
| **② 비교하기** | **특징** | 상대방이나 상황을 다른 사람/경험과 비교하여 비하하거나 우월감을 느끼는 대화입니다. <br> **결과**: 상대의 존재 가치를 떨어뜨리고 열등감, 비난, 수치심을 유발합니다. |
| **③ 책임 회피** | **특징** | '나는 어쩔 수 없었다', '나는 단지 시키는 대로 했을 뿐이다'와 같이 **자신의 행동에 대한 책임을 회피**하는 방식의 대화입니다. <br> **활용 예**: "나는 **~해야만 한다**", "나는 **~할 수밖에 없다**". |
| **④ 요구하기** | **특징** | 부탁이 아닌 **강요**나 **명령**을 하는 대화입니다. <br> 요구가 충족되지 않았을 때 상대방을 **비난하거나 벌을 주겠다는 의도**가 포함되어 있습니다. |

### 2. 비폭력 대화 (NVC: Nonviolent Communication) 모델의 4가지 요소

| 단계 | 개념 | 역할 및 목표 |
| :--- | :--- | :--- |
| **1. 관찰 (Observation)** | 상대방의 말과 행동을 **있는 그대로** 바라봅니다. | 판단이나 평가 없이, 나에게 유익하든 그렇지 않든 **사실**만 서술하여 대화의 객관성을 확보합니다. |
| **2. 느낌 (Feeling)** | 그 행동을 보았을 때 내가 **어떻게 느끼는가**를 말합니다. | 나의 **내면의 감정**을 표현하며, 상대의 행동이 나에게 미친 정서적 영향을 전달합니다. |
| **3. 욕구 (Need)** | 나의 느낌이 내면의 **어떤 욕구와 연결**되는지 찾아냅니다. | 느낌의 근원인 **충족되지 않은 욕구**를 파악하여 자신의 감정에 책임을 집니다. |
| **4. 부탁 (Request)** | 내 삶을 풍요롭게 하기 위해 다른 사람이 해주기 바라는 것을 **구체적으로** 표현합니다. | 강요가 아닌 **선택의 자유**를 부여하며 행동을 요청합니다. |

### 3. 관찰과 평가의 구별 (NVC 1단계)

| 구분 | 주요 개념 | 상세 설명 및 예시 |
| :--- | :--- | :--- |
| **관찰** | **정의** | **언제, 어디서, 누가, 무엇을** 했는지 객관적으로 서술하며, **시간과 공간에 제한**되어 나타납니다. |
| **평가** | **정의** | 관찰한 사실에 **나의 생각, 판단, 해석이 들어간 것**입니다. |
| **구별의 필요성** | **이유** | 관찰을 평가와 섞어 말하면 상대방은 비난으로 받아들이고 **방어적인 자세**를 취하게 되어 문제 해결을 어렵게 합니다. |
| **예시 (관찰)** | OOO은 이번 달에 회의에 한 번도 참석하지 않았다. | 팩트(Fact)만을 말합니다. |
| **예시 (평가)** | OOO은 회의 시간에 항상 게으르다. | '항상', '게으르다'는 나의 주관적 판단입니다. |

### 4. 느낌과 생각의 구별 (NVC 2단계)

| 구분 | 주요 개념 | 상세 설명 및 예시 |
| :--- | :--- | :--- |
| **느낌 (Feeling)** | **정의** | 관찰한 것을 보고 느끼는 **나의 순수한 감정**입니다. (예: 기쁘다, 슬프다, 화난다, 즐겁다, 외롭다, 서운하다, 지친다 등). |
| **생각 (Thinking)** | **정의** | 어떤 것에 대한 **나의 평가, 판단, 해석이 들어간 언어**입니다. <br> **주의**: '느낌이다' 또는 '~처럼 느껴진다'라고 끝나도 사실은 **'생각'**인 경우가 많습니다. |
| **구별의 필요성** | **이유** | 느낌과 생각을 혼동하면 자신의 감정을 제대로 표현하지 못하며, 상대방의 행동에 대한 나의 **평가나 판단**이 들어간 표현은 **갈등을 부추길** 수 있습니다. |
| **예시 (생각)** | **'무시당한 느낌이야'** (→ 사실은 '무시당했다고 생각한다'는 평가). |
| **예시 (순수 느낌**)| **'화난다', '서운하다', '분하다'** (→ 나의 순수한 감정). |


## 📝 12주차: 비폭력 대화(NVC) II 상세 요약 (시험 대비)

### 1. NVC 3단계: 욕구 (Need) 의식하고 표현하기

| 구분 | 주요 개념 | 상세 설명 및 특징 |
| :--- | :--- | :--- |
| **느낌과 욕구의 관계** | **신호등 원리** | **느낌**은 **욕구의 충족 여부**를 알려주는 신호등과 같습니다. <br> - **긍정적 느낌 (그린라이트)**: 욕구가 충족되고 있음. (예: 뿌듯함, 행복감) <br> - **부정적 느낌 (레드라이트)**: 욕구가 좌절되고 있음. (예: 허탈감, 참담함) |
| **느낌의 근원** | **책임 소재** | 나의 느낌(감정)의 근원은 **상대방의 행동**이 아니라, 그 행동에 대한 나의 **욕구 충족/좌절**에 있습니다. <br> NVC는 자신의 감정에 대해 **스스로 책임**지는 것을 중요하게 생각합니다. |
| **욕구의 특징** | **보편성** | 욕구는 보편적이며, 인간이라면 누구나 가지고 있는 기본적인 요소입니다. <br> (예: 안전, 존중, 사랑, 소속감, 인정, 휴식, 의미 있는 삶 등) |
| **욕구 표현의 어려움** | **문제점** | 자신의 욕구를 솔직하게 인식하고 표현하는 것이 성숙한 소통이지만, 현실에서는 욕구 대신 상대를 **비난하거나 평가**하는 말로 대체하여 표현하기 쉽습니다. |
| **욕구 인식 훈련** | **방법** | '나는 ~하기를 원한다' 대신 '**나는 ~한 욕구가 있다**'는 말로 자신의 내면을 들여다봅니다. |

### 2. NVC 4단계: 부탁 (Request) 표현하기

| 구분 | 주요 개념 | 상세 설명 및 특징 |
| :--- | :--- | :--- |
| **부탁의 정의** | **정의** | 내 삶을 더 풍요롭게 하기 위해서 다른 사람이 해주기 바라는 것을 **구체적으로 표현**하는 것입니다. |
| **부탁과 강요의 구별** | **차이점** | 부탁은 상대방에게 **'선택할 자유'**를 부여하지만, **강요**는 상대방이 거절했을 때 **비난이나 처벌**을 예상하게 합니다. |
| **부탁이 아닌 경우** | **판단 기준** | 상대방이 내 부탁에 응하지 않았을 때, 부탁한 사람이 그 거절 이유에 대해 **공감**해주면 그것은 강요가 아닌 **부탁**이었다는 것을 알 수 있습니다. |
| **부탁 시 주의사항** | **구체성** | **'하지 말아 달라'**는 부정적 표현보다는 **'무엇을 해 달라'**는 **긍정적이고 구체적인 행동**을 요청해야 합니다. <br> (예: "내 말 좀 이해해 줘" (X) → "내 이야기를 10분만 들어줘" (O)) |
| **솔직한 반응 부탁하기** | **3가지 질문** | 부탁을 표현한 후, 상대방의 솔직한 반응을 유도하여 부탁이 강요로 느껴지지 않게 합니다. <br> 1. 내가 한 말에 대해 상대가 **무엇을 느끼는지** 묻는다. <br> 2. 상대가 내 제안에 대해 **무엇을 생각하는지** 묻는다. <br> 3. 상대가 내 제안을 **받아들일 의사가 있는지** 확인한다. |
| **부탁 거절에 대한 태도** | **성숙한 관계** | 상대의 거절을 **나에 대한 거절**이 아니라는 것을 구분하여 받아들여야 합니다. <br> '도움을 주면 고맙고, 그렇지 않으면 그만이다'라는 마음가짐이 필요합니다. <br> **거절은 단절이 아니며**, 도움 요청에 거절할 수도, 거절당할 수도 있는 관계가 서로 편안하고 **성숙한 관계**입니다. |
